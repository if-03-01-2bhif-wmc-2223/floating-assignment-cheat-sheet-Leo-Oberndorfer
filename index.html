HTML Cheatsheet

Html and Css

General components are the opening and closing tag and in between is the clement content.
There are also void elements. They don't have an end tag or any content and are also called empty elements. For example: <br>
Attributes provide additional information about elements. Elements can be styled using a css file.
A css rule is divided into 3 parts:
- Selector: Can be an HTML element, an id or a class
- Property: the name of the property you want to style such as `background-color` or `padding`
- Value: value that will be assigned to attribute

A CSS selector can contain more than one simple selector. Between the simple selectors, we can include a combinator.
There are four different combinators in CSS:
descendant selector (space) - selects all elements inside that element
child selector (>) - selects all elements that are children of that element
adjacent sibling selector (+) - selects the first element inside that element
general sibling selector (~) - selects all elements that are next siblings of these elements

With css you can customize almost everything. You can apply different effects to elements and make your webpage look nicer.

Graphic formats

jp(e)g: kann bis zu 16,7 Mio. Farben darstellen und eignet sich daher gut für detaillierte Fotos; kann gut komprimiert werden

gif: nur bis 256 Farben möglich und Animation möglich; verlustfreie Kompression

png: Transparent und Verlustfrei (wie bei jpg 16,7 Mio. farben)

ico: können mehrere Einzelbilder enthalten, die das System je nach benötigter Auflösung und Farbtiefe benutzt

svg: Scalable Vector Graphic; Unterstützt Interaktivität und Animationen

mpg / mp4: Die Bildrate ist eingestellt auf 25 fps.
